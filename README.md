# sa5mchavez

Este proyecto es una aplicación web moderna construida con **Vue 3** y **Vite**, diseñada para ofrecer una experiencia de usuario fluida y eficiente.

## 🚀 Tecnologías Usadas

- **Vue 3**: Framework progresivo de JavaScript para construir interfaces de usuario.
- **Vite**: Herramienta de construcción de próxima generación que ofrece un servidor de desarrollo rápido.
- **Vue Router**: El enrutador oficial para Vue.js.
- **Vue i18n**: Soporte de internacionalización para aplicaciones Vue.

## 📂 Estructura del Proyecto

La lógica principal se encuentra en el directorio `src/`:

- `src/assets/`: Recursos estáticos como imágenes y estilos globales.
- `src/components/`: Componentes de UI reutilizables.
- `src/pages/`: Vistas principales de la aplicación (Home, etc.).
- `src/router/`: Configuración de rutas de la aplicación.
- `src/i18n/`: Archivos de traducción y configuración de idiomas.
- `src/App.vue`: Componente raíz de la aplicación.
- `src/main.js`: Punto de entrada de la aplicación.

## 🛠️ Cómo Ejecutar

### Requisitos Previos

Asegúrate de tener [Node.js](https://nodejs.org/) instalado en tu sistema.

### Instalación

Clona el repositorio e instala las dependencias:

```bash
npm install
```

### Desarrollo

Inicia el servidor de desarrollo con recarga en caliente:

```bash
npm run dev
```

### Producción

Para compilar y minificar para producción:

```bash
npm run build
```

Para previsualizar la compilación de producción:

```bash
npm run preview
```
